# Python Send Email Builtin SMTP

This example demonstrates how to send an email using Python's built-in `smtplib` and `email` modules, without requiring any external libraries. It constructs a simple text email and sends it via an SMTP server, showcasing the power of Python's standard library for email automation.

## Language

`python`

## How to Run

1. Set environment variables: `SENDER_EMAIL`, `SENDER_PASSWORD` (use an app password for Gmail), `RECEIVER_EMAIL`. Optionally, set `SMTP_SERVER` and `SMTP_PORT`.
2. Run the script: `python main.py`
